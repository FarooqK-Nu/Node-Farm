# 🌱 Node Farm Project

A simple Node.js project that serves product data and renders pages using core modules and basic routing.

##  Features

- Displays a list of farm products  
- Dynamic routing for product details
- server side rendering
- JSON-based data handling  
- Simple HTML templates  
- Built using core Node.js modules (no frameworks)


##  Technologies Used

- Node.js  
- HTTP module  
- FS (File System) module  
- URL module  
- Vanilla JavaScript  
- HTML & CSS
